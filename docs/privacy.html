Personvernerklæring

Sist oppdatert: 7. desember 2025

1. Introduksjon

Denne personvernerklæringen gjelder for appen "Mine Våpen". Vi tar ditt personvern på alvor. Denne appen er designet for å lagre data lokalt på din enhet, og vi (utvikleren) samler ikke inn, lagrer eller deler dine personlige data.

2. Datainnsamling og Lagring

Lokal Lagring: All informasjon du legger inn i appen (våpeninformasjon, serienummer, navn, medlemskap osv.) lagres utelukkende i en lokal database (SQLite) på din egen telefon.

Ingen skytjenester: Appen sender ingen data til eksterne servere, skytjenester eller tredjeparter.

3. App-tillatelser

Appen kan be om følgende tillatelser for å fungere optimalt:

  • Lagring/Filer: Dette kreves kun hvis du velger å eksportere data (CSV) eller ta backup av databasen. Appen trenger tilgang for å kunne lagre disse filene på enheten din eller dele dem med andre apper du velger.

4. Backup og Eksport

Appen tilbyr funksjonalitet for å ta backup og eksportere data. Disse filene genereres lokalt. Det er du som bruker som bestemmer hvor disse filene skal lagres eller hvem de skal deles med (f.eks. via e-post eller skylagring som Google Drive).

5. Tredjeparter

Siden appen ikke kobler seg til internett for databehandling, deles ingen brukerdata med tredjeparts annonsører eller analysetjenester.
